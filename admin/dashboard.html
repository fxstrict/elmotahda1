مصدر البيانات الحالي

ملاحظة مهمة حول طريقة عمل اللوحة

هذا الموقع موقع ثابت (Static) بدون سيرفر أو قاعدة بيانات، لذلك تعمل لوحة التحكم بالطريقة التالية:

  • أي تعديل (منتج، تصنيف، صورة) يُحفظ في متصفحك الحالي فوراً، وتراه على الموقع مباشرة في نفس المتصفح.
  • هذا التعديل لا يظهر تلقائياً للزوار الآخرين لأن الموقع مستضاف كملفات ثابتة (مثل GitHub Pages).
  • لجعل التعديلات ظاهرة للجميع بشكل دائم: اذهب لتبويب "تصدير / استيراد"، نزّل ملفات JSON المحدَّثة، ثم استبدل بها الملفات في مجلد data/ داخل مستودع الموقع وارفعها (Commit + Push).

تصدير / استيراد البيانات

لحفظ التعديلات بشكل دائم على الموقع المنشور

1) تصدير الملفات المحدَّثة

نزّل ملفي JSON التاليين، ثم استبدل بهما نفس الملفين داخل مجلد data/ في مشروع الموقع، وارفعهما (Commit + Push) حتى تظهر التغييرات لكل الزوار.

2) استيراد ملفات JSON

إذا كان لديك ملف products.json أو categories.json جاهز (مثلاً تم تعديله بأداة Excel)، ارفعه هنا ليحل محل البيانات الحالية في اللوحة والموقع (لهذا المتصفح).

3) إعادة الضبط

حذف كل التعديلات المحفوظة في هذا المتصفح والعودة لبيانات الموقع الأصلية (data/products.json و data/categories.json).
